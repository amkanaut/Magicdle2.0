# Magicdle 2.0

Magicdle is a daily guessing game for Magic: The Gathering cards.

## Project Structure

- `/backend`: Node.js + Express API for fetching card data from Scryfall and managing daily challenges.
- `/frontend`: React + Vite single-page application for the game interface.

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. Create a `.env` file based on `.env.example`.
4. `npm start` or `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
